EAPI Egueb_Dom_Node* egueb_dom_event_relative_get ( Egueb_Dom_Event *  thiz)

Get the node relative to the current target.

Parameters:
[in]thizThe event to get the relative node from
Returns:
The relative node

For events that are on the EGUEB_DOM_EVENT_PHASE_AT_TARGET phase the relative node is NULL. For events that on the EGUEB_DOM_EVENT_PHASE_CAPTURING phase the relative node is the child node where the event will propagate later. For EGUEB_DOM_EVENT_PHASE_BUBBLING phase the relative node is the child node where the event propagated before.
